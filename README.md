> This is a Microverse Ruby and Rails Project

> Members Only!

## Description

The purpose of this project is learning how to use authentication by using devise gem in rails.

## Features

In this project, the following actions are done;

- Created and migrated a User model of devise
- Created and migrated a Post model with body text
- Added buttons to sign in and sign up 
- Added styles by using Bootstrap

## Built With

- Ruby on rails
- Devise
- Git
- Shields.io

## Setup

- Clone this repository
- Open terminal
- Change directory by using `cd members-only/`
- Run `bundle install`
- Open rails server by using `rails s`
- Open browser and go to http://localhost:3000
- Sign in by using any username and email
